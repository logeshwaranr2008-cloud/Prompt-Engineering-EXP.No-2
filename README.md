EX-02-Cross-Platform-Prompting-Evaluating-Diverse-Techniques-in-AI-Powered-Text-Summarization

AIM:

To evaluate and compare the effectiveness of prompting techniques (zero-shot, few-shot, chain-of-thought, role-based) across different AI platforms (e.g., ChatGPT, Gemini, Claude, Copilot) in a specific task: text summarization.

SCENARIO:

You are part of a content curation team for an educational platform that delivers quick summaries of research papers to undergraduate students. Your task is to summarize a 500-word technical article on "The Basics of Blockchain Technology" using multiple AI platforms and prompting strategies.

Your goal is to determine which combination of prompting technique + platform provides the best summary in terms of:

1.Accuracy

2.Coherence

3.Simplicity

4.Speed

5.User experience

OUTPUT:

Zero-shot   Prompting

| Platform    | Output

| **ChatGPT** | Blockchain is a digital system that records transactions across many computers so that the data cannot be easily changed. Each record is stored in blocks connected together using cryptography. Blockchain improves transparency and security but still faces issues such as high energy use and slow scalability. |

| **Gemini**  | Blockchain is a distributed ledger technology where transactions are stored in blocks connected in a chain. It powers cryptocurrencies and can also be used in industries like healthcare, finance, and logistics. Despite its advantages, challenges such as regulation and energy consumption remain.             |

| **Claude**  | Blockchain is a decentralized record-keeping system that stores data in linked blocks across many computers. Because each block is connected with cryptographic hashes, altering data becomes extremely difficult.  |

Few-shot Prompting

| Platform    | Output                                                                                                                             |

| **ChatGPT** | Blockchain is a decentralized system that securely records transactions across multiple computers without a central authority.     |

| **Gemini**  | Blockchain is a distributed ledger that securely stores transaction data across a network and removes the need for intermediaries. |

| **Claude**  | Blockchain is a secure decentralized ledger that records transactions in linked blocks across a network.                           |

Additional Summary

| Platform    | Output                                                                                                                                                                                                                        |

| **ChatGPT** | Blockchain is a distributed digital ledger that records transactions in blocks linked together using cryptography. Because the ledger is shared across many computers, it is difficult to alter records once they are stored. |

| **Gemini**  | Think of blockchain as a shared digital notebook where every transaction is recorded permanently. Each page (block) is connected to the previous one using encryption.                                                        |

| **Claude**  | Blockchain can be understood as a decentralized database where information is stored in blocks that form a secure chain. Because many computers maintain the ledger, it becomes extremely difficult to change past data.      |

| **Gemini**  | 4        | 4         | 5          | 5     | 4               |
